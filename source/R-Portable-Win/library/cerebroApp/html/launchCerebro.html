Launch Cerebro interface.

Description

Launch Cerebro interface.

Usage

launchCerebro(version = "v1.2", maxFileSize = 800)

Arguments

version

Which version of Cerebro to launch: "v1.0", "v1.1", "v1.2". Defaults to "v1.2".

maxFileSize

Maximum size of input file; defaults to 800 MB.

Details

Launch Cerebro interface.

Value

Shiny application.

Examples

if ( interactive() ) {
  launchCerebro(maxFileSize = 800)
}
